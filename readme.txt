== Description ==
Simple Lightbox implementation using responsive lightbox 'Swipebox' implementation for the GD bbPress Toolbox Pro attachments images.

== Installation ==
= General Requirements =
* PHP: 7.0 or newer
* GD bbPress Toolbox Pro: 6.5 or newer
* bbPress 2.6.2 or newer
* WordPress: 5.1 or newer

= PHP Notice =
* Plugin doesn't work with PHP 5.6 or older versions.

= WordPress Requirements =
* WordPress: 5.0 or newer
* bbPress 2.6.2 or newer

= WordPress Notice =
* Plugin doesn't work with WordPress 5.0 or older versions.

= Basic Installation =
* Plugin folder in the WordPress plugins folder must be `gd-bbpress-toolbox-lightbox`.
* Upload `gd-bbpress-toolbox-lightbox` folder to the `/wp-content/plugins/` directory.
* Activate the plugin through the 'Plugins' menu in WordPress.
